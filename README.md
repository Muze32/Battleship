# Battleship 🚢

A browser-based Battleship game built with vanilla JavaScript, featuring a CPU opponent with randomized moves and a two-player mode.

🔗 **[Play it here](https://muze32.github.io/Battleship/)**

---

## Features

- Play against the CPU or a second player
- Randomized ship placement at the start of each game
- CPU opponent with a random delay to simulate thinking
- Turn-based gameplay with board visibility control for two-player mode
- Reset and randomize buttons for quick restarts

---

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+ Modules)
- Webpack
